Selected Projects

Road Roughness Data Management Platform

I am reponsible for the entire development for this web application, including both frontend and backend. This platform is used to track, edit, and download the road data collected by raspberry-pi.

Node.js and Express are used for backend. The web application is hosted on AWS server. So users can access it from anywhere with the internet connection. MySQL is used for database.

GitHub Link: https://github.com/irisma00/road-roughness
